## 🌐 Размещение на хостинге

Выберите один из вариантов:

### Вариант 1: Netlify (Рекомендую! Самый простой)

#### Способ A: Drag & Drop (без Git)

1. **Зайдите на** https://netlify.com
2. **Зарегистрируйтесь** (бесплатно)
3. Перейдите в **"Sites"**
4. **Перетащите папку** `OrderAppPWA` в окно браузера
5. **Готово!** Получите ссылку типа `random-name-123.netlify.app`

#### Способ B: Через Git (для автообновлений)

```bash
# 1. Установите Git (если нет)
# Скачайте: https://git-scm.com/download/win

# 2. В папке OrderAppPWA откройте PowerShell и выполните:
git init
git add .
git commit -m "Initial commit"

# 3. Создайте репозиторий на GitHub

# 4. Загрузите код:
git remote add origin https://github.com/ваш-username/orderapp.git
git push -u origin main

# 5. На Netlify:
# - New site from Git
# - Выберите GitHub
# - Выберите репозиторий
# - Deploy!
```

## 🎯 Использование

### Первый запуск

2. Подождите 2 секунды (анимация загрузки)
3. Вы увидите главный экран с 5 вкладками внизу

### Загрузка прайс-листа

#### Способ 1: Автоматически из Telegram (рекомендуется)

1. Перейдите в **Настройки** (последняя вкладка)
2. Нажмите **"Обновить прайс-лист"**
3. Приложение попытается загрузить файл от бота @postavuglipricebot
4. Подождите несколько секунд

#### Способ 2: Вручную (если автоматически не работает)

Добавьте кнопку загрузки в настройки:

1. Откройте файл `telegram.js` в редакторе
2. Найдите функцию `renderSettings`
3. Добавьте кнопку:
```javascript
<button class="btn btn-secondary" onclick="uploadExcelFile()">
    📄 Загрузить файл вручную
</button>
```

Теперь в настройках будет кнопка для выбора файла с устройства.

### Работа с каталогом

1. **Поиск** - введите название товара в строку поиска
2. **Фильтры** - нажмите кнопку "Фильтры" для выбора категории/бренда
3. **Добавить в корзину** - нажмите синюю кнопку "+"

### Оформление заявки

1. Перейдите в **Корзину** (вторая вкладка)
2. **Выберите заведение** (Мята или Френдс)
3. **Проверьте товары** и количество
4. Нажмите **"Отправить заявку"**
5. Подтвердите отправку
6. Готово! Заявка отправлена менеджеру в Telegram

### Просмотр истории

1. Перейдите в **История** (третья вкладка)
2. Увидите все отправленные заявки
3. Нажмите на любую для просмотра деталей

### Аналитика

1. Перейдите в **Аналитика** (четвертая вкладка)
2. Увидите:
   - Общую сумму расходов
   - Количество заявок
   - Средний чек
   - Расходы по месяцам
   - Статистику по заведениям

## 🐛 Решение проблем

### Проблема: Приложение не устанавливается

**Причина:** Сайт не на HTTPS или Service Worker не работает

**Решение:**
1. Убедитесь что URL начинается с `https://` (не `http://`)
2. Проверьте консоль браузера (Safari → Разработка → Консоль)
3. Убедитесь что файл `sw.js` доступен

### Проблема: Не загружается прайс-лист

**Причина:** Бот не отправил файл или токен неверный

**Решение:**
1. Убедитесь что бот @postavuglipricebot работает
2. Напишите боту команду для получения прайса
3. Проверьте токен в файле `telegram.js`
4. Попробуйте загрузить файл вручную

### Проблема: Заявка не отправляется

**Причина:** Нет интернета или проблема с Telegram API

**Решение:**
1. Проверьте интернет-соединение
2. Откройте консоль браузера - посмотрите ошибки
3. Проверьте что ID менеджера правильный: `1046604055`
4. Попробуйте позже (возможны временные проблемы с Telegram)

### Проблема: Данные пропали

**Причина:** Очистка кеша браузера или переустановка

**Решение:**
- Данные хранятся локально в браузере
- При очистке Safari всё удаляется
- Решение: регулярно экспортируйте данные (функцию можно добавить)

### Проблема: Приложение не обновляется

**Причина:** Service Worker кеширует старую версию

**Решение:**
1. Закройте приложение полностью
2. Откройте Safari
3. Зайдите в Settings → Safari → Clear History and Website Data
4. Откройте сайт заново
5. Переустановите приложение

1. **Favicon.io**: https://favicon.io/favicon-generator/

   - Сервис создаст все размеры

### Без иконок (временно)

Если не хотите заморачиваться:

В `index.html` закомментируйте строки с иконками:
```html
<!-- <link rel="apple-touch-icon" sizes="180x180" href="icons/icon-180.png"> -->
```

Приложение будет работать без иконки.

## 🔄 Обновление приложения

### Если используете Netlify/Vercel

Просто загрузите новые файлы - обновится автоматически!

### Если используете свой хостинг

1. Загрузите новые файлы через FTP
2. Пользователи получат обновление при следующем запуске

### Кеш Service Worker

При обновлении измените версию в `sw.js`:
```javascript
const CACHE_NAME = 'orderapp-v2'; // Было v1, стало v2
```

Это заставит браузер обновить кеш.

Добавьте функцию экспорта в настройки:
```javascript

```

Это сохранит JSON файл со всеми заказами.
